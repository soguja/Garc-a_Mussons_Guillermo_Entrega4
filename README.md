# Monta un PC – Videojuego educativo

Videojuego educativo desarrollado con Construct 3 donde el jugador debe montar correctamente un ordenador arrastrando y colocando sus componentes en la caja del PC.

## Objetivos
- Introducir el uso de editores de juegos web
- Comprender las bases del desarrollo 2D
- Desarrollar habilidades básicas de programación
- Publicar un juego online

## Mecánica de juego
El jugador debe arrastra los componentes del PC y los coloca dentro de la caja.  
Cuando todos los componentes están correctamente colocados, aparece un mensaje final.
